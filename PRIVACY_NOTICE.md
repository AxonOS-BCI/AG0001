# Privacy Notice

AG0001 / Chronos Boundary Flight is a local-only browser game.

It does not intentionally collect:

- accounts;
- names;
- email addresses;
- biometric data;
- raw neural data;
- wallet addresses;
- analytics identifiers;
- advertising identifiers;
- telemetry;
- location data.

The game may store only a local high score under this browser-local namespace:

```text
axonos_ag0001_chronos_
```

No data is uploaded by this static GitHub Pages build.
