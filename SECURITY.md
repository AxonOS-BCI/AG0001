# Security Policy

## Supported version

`v0.1.0`

## Public runtime exclusions

This repository does not include:

- backend services;
- account system;
- telemetry SDK;
- wallet connection;
- payment processing;
- real BCI driver;
- neural data ingestion;
- hardware stimulation control;
- third-party scripts or CDNs.

## Reporting

Report security issues privately to:

```text
security@axonos.org
```
