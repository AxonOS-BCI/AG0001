# AG0001 v0.1.0 — Chronos Boundary Flight

## Summary

First playable AG0001 release: a premium upward-flight browser game that expresses the AxonOS thesis through movement, pressure, consent, Privacy Vault containment, Zero Trust probes, and typed-intent release.

## What changed

- Added full static GitHub Pages game.
- Added custom canvas flight engine.
- Added deterministic seed-based route generation.
- Added aircraft physics: acceleration, drag, roll banking, boost heat and shield braking.
- Added AxonOS metrics: Boundary Integrity, Privacy Vault, Consent Coherence, Cognitive Flow, Latency Pressure, Raw Leak Risk, Stimulation Risk.
- Added five scenarios.
- Added mobile touch controls.
- Added muted-by-default Web Audio engine hum.
- Added local-only high score.
- Added documentation, privacy/security notices, claims matrix, IP notice and verification script.
- Added CI and GitHub Pages workflows.

## Release invariants

- No telemetry.
- No backend.
- No wallet connection.
- No payment to play.
- No third-party scripts or CDNs.
- No real neural data.
- No medical/clinical claims.
